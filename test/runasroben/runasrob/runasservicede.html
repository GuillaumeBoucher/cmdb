Run as Service

Starte Programm als Dienst nach dem Systemstart mit höchsten Rechten, bevor ein Benutzer angemeldet ist

  • Hilfreich für Hintergrundaufgaben wie Monitoring, Kopieraktionen oder Prüfroutinen auf einem Computer.
  • Nützlich wenn die Anwendung mit höchsten Rechten starten soll und dabei keine Benutzeraktion erforderlich ist.
  • Das Programm muss seine Arbeit erledigen ohne irgendeine Benutzereingabe, weil es als Windows Dienst im Hintergrund läuft.

Schnellanleitung:

  1. Starten Sie RunAsService.exe als lokaler Administrator.
  2. Drücken Sie den Button >> Install RunAsRob <<.
  3. Wählen Sie über >> Add application << die Anwendung aus, welche als Dienst laufen soll.
  4. Fertig.
  5. Nach einem Neustart des Computers (oder des Dienstes RunasRob) startet die Anwendung als Dienst mit Systemrechten.
